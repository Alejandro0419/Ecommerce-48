## Description

Backend Server of a virtual store with Node.js, Nest.js, and PostgreSQL, with complete documentation done with Swagger Open API

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
